# archlinux picom-ftlabs configuration

## instalation guide
* install picom-ftlabs-git using aur helper
```
yay -S picom-ftlabs-git
```
* install depences
```
sudo pacman -Sy alacritty i3-gaps feh polybar rofi
```
* move **NERDFONTS** to /usr/share/fonts/
* rename **name_this_.backgrounds** to **.backgrounds** and move to home dir
* move all except **NERDFONTS name_this_.backgrounds** to **~/.config/**
